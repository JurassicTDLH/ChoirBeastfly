# ChoirBeastFly

Face the Beastfly tamed by the Choir of the Citadel! 
This mod adds a powerful Beastfly with silk abilities at the The Forum. 

How to Fight:
Play your Needolin at the right side (where you enter the scene) of The Forum and the boss will be summoned. 
It will stay there until being defeated or you go back to the menu.
You can summon it by the same way immediately after you defeat it and leave the scene.

*Try Elegy of the Deep if you need more challenge in Act 3.

How to Use:
1.Download and install BepInEx
2.Ensure that the game has run at least once so that a plugins folder is generated in the BepInEx folder besides the game executable.
3.Download the .zip file from one of the sources listed above.
4.Extract the .zip file to the generated plugins folder.
5.Launch the game and enjoy!
